# Readme

## Installing the Necessary Tools

### Web

- `cd web`
- `npm i`

### Electron

- `cd electron`
- `npm i`

## Running the Apps

There are two different apps that can be run

1. The webapp
    - `npm start`
2. The electron app
    - `npm electron-build`
        - This builds the most recent version of the app
    - `npm electron`
        - Runs whatever version was last built
